## 📋 快速开始

### 第一步：准备成员信息
打开 `teamMembersTemplate.md` 查看详细模板，按照以下任一格式提供31位成员的信息：

**简单格式（推荐）：**
```
1. 张三 - 创意总监 - member-1.jpg
2. 李四 - 设计主管 - member-2.jpg
3. 王五 - 高级设计师 - member-3.jpg
...依此类推到31
```

### 第二步：准备图片
1. 将31张成员照片放入：`public/images/team/` 文件夹
2. 图片命名建议：`member-1.jpg`, `member-2.jpg` ... `member-31.jpg`
3. 推荐尺寸：500x750px（2:3比例）

### 第三步：提供给我
把整理好的成员名单（格式如上）发给我，我会帮你：
- ✅ 更新 `teamMembers.ts` 文件
- ✅ 自动匹配图片路径

- ✅ 完成页面集成
