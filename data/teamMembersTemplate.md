# 团队成员信息模板

请按照以下格式提供31位成员的信息，然后我帮你更新到网站上：

## 填写方式（任选其一）：

### 方式1：表格格式（推荐）
```
编号 | 姓名 | 职位 | 图片文件名
-----|------|------|------------
1    | 张三 | 创意总监 | member-1.jpg
2    | 李四 | 设计主管 | member-2.jpg
3    | 王五 | 高级设计师 | member-3.jpg
...继续到31
```

### 方式2：逐行格式
```
1. 张三 - 创意总监 - member-1.jpg
2. 李四 - 设计主管 - member-2.jpg
3. 王五 - 高级设计师 - member-3.jpg
...继续到31
```

### 方式3：JSON格式
```json
[
  {"id": 1, "name": "张三", "position": "创意总监", "image": "member-1.jpg"},
  {"id": 2, "name": "李四", "position": "设计主管", "image": "member-2.jpg"},
  ...
]
```

## 图片要求：
- 建议尺寸：500x750px（2:3比例）
- 格式：jpg 或 png
- 将图片放在：`public/images/team/` 文件夹
- 命名建议：member-1.jpg, member-2.jpg ... member-31.jpg

## 职位参考（可自定义）：
- 创意总监
- 设计总监
- 高级设计师
- 设计师
- UI/UX设计师
- 视觉设计师
- 动态设计师
- 产品设计师
- 前端工程师
- 等等...
